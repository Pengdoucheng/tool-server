TIME · DISCIPLINE

用一個月的視角，排好你的時間節奏。

這是單人版行事曆：每一天都可以新增多個任務、用不同顏色標記用途。 藍色也可以是「耍廢時間」，重點是你看得懂自己的節奏。

顏色標記說明（自由使用）
只是建議，你可以自己決定每種顏色代表什麼。
紅色：Deadline / 很重要的事情
橘色：學校 / 報告 / 考試
黃色：生活碎事 / 雜事
綠色：運動 / 健康 / 睡眠
藍色：休息 / 耍廢 / 社交
紫色：自我成長 / Side Project
灰色：通勤 / 等待 / 不好分類的時間
每一格日期下方都有「輸入框 + 顏色 + 儲存」，按儲存後會寫進 Supabase。
